# Web Animations API - Course Projects

This repository contains various projects and exercises focused on the Web Animations API.

## Projects & Exercises

- **Animations**: Basic animation examples.
- **Exercise 1**: Initial animation exercises.
- **Exercise 3**: Advanced animation exercises.
- **Scroll Animations**: Animations triggered by scrolling.
- **Timing Functions**: Exploration of animation timing functions.
- **Transitions**: CSS Transitions examples.
- **View Transitions**: Examples using the View Transitions API.
- **Web API**: Direct usage of the Web Animations API.
- **Words Game**: A game implementing web animations.

## Usage

Open `index.html` in your browser to view the list of projects and navigate to them.
